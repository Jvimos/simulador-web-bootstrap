¿Qué ocurre al ingresar una URL?

Cuando el usuario escribe una URL en el navegador, comienza un proceso de comunicación entre el cliente y el servidor mediante el protocolo HTTP. El servidor procesa la solicitud y devuelve los recursos necesarios para mostrar correctamente la página web.
